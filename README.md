# 課題10：PHPで「所属選手一覧」のページを作る

## ①課題内容（どんな作品か）
- 所属する少年サッカーチームでは、今も新入部員が入り続けています。後から入ってきた子やその保護者からすると、全員の顔と名前を一致させるのが大変。
- そこでサッカーチームの選手一覧画面を作りました。
- 選手の写真と情報（名前など）が一覧で見られます。
- Loginすると画面編集ができるようになります（新規選手登録と既存選手の編集）
- Loginしていない人は単に情報を見るだけ。
